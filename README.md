# Aditya-ds-1806 Tap

## How do I install these formulae?

`brew install aditya-ds-1806/tap/<formula>`

Or `brew tap aditya-ds-1806/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "aditya-ds-1806/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
